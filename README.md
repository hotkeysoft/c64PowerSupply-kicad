# C64 Power Supply
Rev A June 2021

***Work in progress, do not use***

### Simple Switching Power Supply
- 120AC Input
- 9VAC Output 1.5A Max w/Fuse
- 5VDC Output 1.5A Max w/Fuse
- LM2596T-ADJ for +5V output
- +5V Adjustable with trimmer
